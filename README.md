# Championship DataHint App

iPhone-venlig PWA til EFL Championship datahint: 1, X eller 2.

## Upload til GitHub

Upload alle filer i denne mappe til roden af dit repository.

## GitHub Pages

1. Gå til Settings
2. Pages
3. Vælg GitHub Actions som source
4. Gå til Actions
5. Kør workflowet `Update Championship data` manuelt første gang

## iPhone

Åbn Pages-linket i Safari og vælg Del > Føj til hjemmeskærm.
